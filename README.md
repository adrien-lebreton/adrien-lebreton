### Selected work

**ogmah** (Fractional CTO, current)
Foodtech SaaS. Built a restaurant cost-optimization platform from scratch, with forecasting at the center: sales prediction, predictive ordering, stock analysis. Result: -15% food waste, -25% admin, +7% margins.
`TypeScript` · `NestJS` · `React` · `PostgreSQL` · `AWS`

**VGP field inspections** (Tech Lead)
Offline-first mobile. A React Native app for technicians running regulatory equipment inspections on job sites. It works with no signal at all and syncs back to the server the moment the network returns. Local SQLite store, secure auth, and a sync layer that resolves conflicts.
`React Native` · `NestJS` · `SQLite` · `offline-first`

**CAFAT &amp; OPT** (New Caledonia)
Public-sector platforms. Client portals and internal tools for a social-security fund (CAFAT) and a postal/telecom operator (OPT): an OpenID SSO on OKTA + Spring Boot, a parcel-distribution app with barcode scan and on-screen signature (SIDO), plus billing tooling.
`Angular` · `Spring Boot` · `OKTA` · `REST`
